# Изменения

## Интерфейс
- Обновлены тексты первых двух экранов сценария.
- Добавлен экран выбора первой автоматической мысли с четырьмя вариантами ответа.
- Добавлен экран «Ход мыслей» после выбора первой мысли.
- Добавлена заглушка перехода к этапу сбора контекста.

## Визуальная часть
- Убраны служебные заголовки из первых двух экранов.
- Выбранная пользователем мысль отображается как основной визуальный акцент на экране «Ход мыслей».
- Упрощена визуальная иерархия карточки комментария.

## Техническая часть
- Расширено локальное состояние экранов до шагов `choice`, `thought` и `context`.
- Добавлены данные вариантов первой автоматической мысли и тексты комментариев.
- Добавлены стили для списка выбора, выбранной мысли и вспомогательных подписей.
